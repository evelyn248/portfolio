Proyecto 1: Highlight Nucleotide

Este programa en Python permite escanear una secuencia de ADN desde un archivo FASTA y resaltar las secuencias de tres nucleótidos que coincidan con la especificada por el usuario. También cuenta cuántas veces aparece la secuencia especificada.

Proyecto 2: Sorting Fasta Files

Este script en Python procesa todos los archivos FASTA en una carpeta especificada, cuenta la cantidad de adeninas (A) en cada secuencia de ADN y ordena las secuencias en orden descendente según el conteo de adeninas.
